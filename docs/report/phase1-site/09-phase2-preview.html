Phase 2 結果：端到端模擬器

Phase 1 的元件已串成端到端模擬器。整合層 M3（事件引擎）與 M6（排程器）自 Phase 2.1 起已實作（不再是「contract-only 預覽」）。本頁是 Phase 2 的系統級結果，每個數字都從 committed validation/reports/phase2/*.json 以 {{key}} 注入。逐步建構過程見下一章「Phase 2 逐步詳解」。誠實分級：measured（真 silicon）/ simulated（無 silicon 的機制）/ extrapolated（超出量測包絡）/ counterfactual（硬體實際跑不了）。

§0唯一硬 silicon 門檻：L4 decodemeasured · ADR-0006

AllCim decode tok/s vs 真實 Metis Card是整個 Phase 2 唯一的硬 silicon 驗證門檻（≤15%）。機制路徑用獨立定價（CIM-GEMV L1 micro-benchmark + mem_lpddr4x 24.2 GB/s 記憶體牆 + 顯式 support/attention/kv 流量），不含任何 e2e-fitted 頻寬：

§1Topology A/B/C 對照：記憶體放在哪，差多少card measured · alpha counterfactual · edge simulated

同一 decode workload，只換「權重住在哪、走什麼通道」。decode 由記憶體牆主導（CIM 運算 ~99% idle），所以 tok/s 完全由記憶體牆決定：

結論（量級才是發現，排序是 by-construction）。沒有自帶 DRAM、權重走 PCIe（3.9 GB/s）會慢 5.7×——這就是「近記憶體」對 decode 的價值。整合進 SoC 用 LPDDR5 比 on-card LPDDR4x 快 1.208×。殘量警示（#58）：A 的 24.2 GB/s 只在模型塞得進 16 GiB on-card DRAM 時成立；溢出會退化成 A・無 DRAM 的 host-PCIe 牆（spill 未建模）。
§2敏感度：結論對 ±20% 不確定性穩健嗎simulated · ADR-0006 L5

對 AllCim serial 路徑唯一雙邊有效的旋鈕 eff_BW 做 ±20%（釘在 24.2 錨：19.36 ↔ 29.04 GB/s）。energy 報 ±20% band 不報點。knee / interconnect_efficiency 只在 k>1 競爭路徑雙邊有效（SIMULATED，排除在 serial 結論外）。

結論穩健性：通過 —— decode 為頻寬主導（tok/s 隨 eff_BW 單調變化）、模型尺寸排序 1B>3B>8B 在整個 band 內維持。

§413B/32GB 級外推（Qwen2.5-14B）extrapolated · NOT validated

Qwen2.5-14B（≈14B，超 8B 包絡一倍，代表 ADR-0006 的 ~13B 延伸點）經 ADR-0007 FakeTensor meta-trace 餵同一引擎外推——非支援設定、無 14B silicon。三個外推維度都標記：

  • 模型尺寸（>8B）：引擎預測 1.52 tok/s。
  • CIM 形狀：241 / 337 個 per-decode CIM matmul 形狀超出 M1 native 量測包絡（floor 外推）——不只是「模型更大」。
  • 容量（>16 GB）：釘死公式 resident_footprint = weights+KV+activation = 14.11 GB；在 16 GiB 卡上 ctx 1024 尚餘 1.89 GB，但 ctx ~18471 即溢出 → spill 到 host PCIe（未建模，#58）。
§5混合精度：限制（不是已驗證的加速）simulated · LIMITATION · D3

專案的混合精度設定＝CIM-INT8 matmul × GPU-FP16 attention。Phase 2 只證 conversion-op（精度邊界 dequant/requant）的成本（8B 每 decode token 96 次 cast）+ 模擬的異質 decode 結構。

限制（明列，非加速宣稱）。(1) 輸出品質未在目標 silicon 量測（D3）——v1 只證成本，不證「混精度等準度」；品質須引用文獻、列為 limitation。(2) CimHetero 受 GPU attn_bmm_us（未最佳化下界）限制，比 AllCim 慢（8B 約 59% of AllCim）——價值在 conversion 成本 + 忠實結構，非加速。(3) 無並發單元 silicon（Aetina 送修 #52）→ SIMULATED，非 validated。
