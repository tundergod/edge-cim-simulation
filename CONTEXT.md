## Glossary

- **CIM** â€” Compute-in-Memory. The study unit; on Metis it is **D-IMC** (Digital In-Memory Compute), INT8.
- **AIPU** â€” Axelera's quad-core Metis accelerator (the CIM device).
- **Metis Alpha** â€” pre-production M.2 on the Aetina board; no on-card DRAM; cannot compute LLM (`-1301` wall). Used for CIM/PCIe/NPU/GPU/CPU micro-benchmarks.
- **Metis Card** â€” production card that runs precompiled INT8 LLMs; decode is a ~24 GB/s on-card-DRAM memory wall. Source of the L4 anchor.
- **prefill / decode** â€” the two LLM inference phases. prefill = compute-bound (GEMM); decode = memory-bound (GEMV). The CIM compute-boundâ†”memory-bound split is the core axis.
- **unit** â€” a compute unit in the simulated SoC: CIM / NPU (RKNPU2) / GPU (Mali) / CPU (A76).
- **M1â€“M8** â€” simulator modules (M1 CIM tile, M2 memory, M3 event engine, M4 NPU/GPU/CPU, M5 workload/trace gen, M6 scheduler/mapper, M7 energy, M8 thermal-optional). See `OVERALL.md`.
- **L1â€“L6** â€” cross-validation layers (what the simulator is checked against). See `OVERALL.md`.
- **op inventory** â€” the distinct op types + shape parametrization a model executes (Phase 0.1 output).
- **trace** â€” the ordered per-token opĂ—shape stream for a (model, prefill_len, decode_len) run.
- **workload** â€” the actual inputs (tasks/datasets + length profiles), not just the model. Layer A: ShareGPT / GSM8K / LongBench-TriviaQA / HumanEval. Layer B: synthetic (prefill Ă— decode) sweep.
- **Phase 0.1 / 0.2 / 0.3 / 1 / 2** â€” trace+op gen / board measurement (no temp) / thermal / per-component equation-fit & validation / simulator integration.
- **honesty tiers** â€” the provenance label every number must carry (CLAUDE.md Â§ Honesty). **measured** = traces to committed silicon JSON (the only tier allowed validation language). **simulated** = a mechanism run with no silicon ground truth (plausible-but-unmeasured; e.g. the contention knee, the edge topology). **extrapolated** = the validated mechanistic model run BEYOND its measured envelope (e.g. >8B model size, >16 GB capacity, per-op CIM shapes past the M1 native envelope). **counterfactual** = a number for hardware that physically *cannot* run the workload (Metis Alpha: `-1301` closed firmware + no on-card DRAM) â€” strictly stronger than simulated (the device is impossible, not merely unmeasured).

Architectural decisions, once made, are recorded as ADRs under `docs/adr/` (created lazily).
